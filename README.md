# Resumo das Aulas
Aula __teórica__ (24/02/2026) e __prática__ (26/02/2026)
- Aula teórica
  
  No início da aula, o professor falou sobre diversos tópicos importantes, tais como arquiterura de um site, uma breve introduçao a versionamento e deploy, introdução ao JavaScript, servidores e hospedagem, entre outros.
  
  Após isso foi realizada uma apresentação de todos os alunos presentes e do professor. Na apresentação todos falaram o nome, se já tem experiência na área, expectativas para quando finalizar o curso e hobbies.
  
- Aula prática
  
  Na aula prática, foi pedido para que nós formassemos grupos de 3 até 5 integrantes, para escolhermos 3 sites e realizarmos uma pesquisa sobre eles. De maneira resumida, na pesquisa, iremos mostrar estatísticas e tecnologias utilizadas no site.
  
  Para as estatísticas utilizamos [SimilarWeb](https://chromewebstore.google.com/detail/similarweb-website-traffi/hoklmmgfnpapgjgcpechhaamimifchmp), para mostrar as tecnologias, utilizamos [WhatRuns](https://www.whatruns.com/).
