# Desenvolvimento-Web

Na aula de hoje, os alunos aprenderam sobre algumas tecnologias de alguns sites, realizando um determinado tipo de busca por site, logo após usando uma extensão ao seu favor, chamada de "Whats Run" para identificação das tecnologias utilizadas.
Após essa tarefa, os alunos deveriam pesquisar sobre cada uma e qual a utilidade das mesmas, e deveria criar um esquema digital utilizando a plataforma "Miro".
O meu grupo realizou a tarefa por completo, e fez um esquema parecido com um "Mapa Mental" para melhor vizualização.
